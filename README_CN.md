# MLX-Audio

基于Apple MLX框架的文本转语音(TTS)和语音转文字(STT)库，专为Apple Silicon优化。

## 🚀 两个版本

### 🌐 Web UI版本
- **适合开发者**：完整API访问，可定制，可扩展
- **安装方式**：克隆仓库 + 运行脚本
- **大小**：约2GB（模型单独下载）
- **最适合**：开发、集成、定制

### 📱 Mac App版本  
- **适合终端用户**：一键安装，无需配置
- **安装方式**：下载592MB All-in-One DMG
- **大小**：592MB（包含模型）
- **最适合**：快速使用，非技术用户

## ✨ 功能特性

- 🌊 实时流式播放（首播<500ms）
- 🎨 动态频谱可视化
- 🌍 12种语言UI
- 🎤 54个声音，9种语言
- 📊 实时性能监控

## 🎯 快速开始

### Web UI版本
```bash
./scripts/setup.sh
./scripts/start.sh
open http://localhost:8002
```

### Mac App版本
1. 下载592MB All-in-One DMG
2. 拖拽到Applications
3. 启动即用
